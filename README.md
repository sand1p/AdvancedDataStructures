# AdvancedDataStructures

## How to run 
**Build tool used: sbt** 
 - Enter project directory CleanStrikeGame*
 - Open sbt console,
    execute *sbt* command 
 - Run application, execute *run* command
 - Run unit test cases, execute *test* command
        
## Tech Stack:     
  - Scala
  - sbt 
  - IDE: Intellij
